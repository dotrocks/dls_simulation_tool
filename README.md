# dls_simulation_tool

## Usage

- Install [Rust](https://rust-lang.org/learn/get-started/)
- Clone this repository
- Clone submodules:

  ```bash
      git submodule update --init --recursive
  ```

- Build the project:

  ```bash
      cargo build --release
  ```

- Run the simulation tool:

  ```bash
      cargo run --release
  ```
